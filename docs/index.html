LibreELEC Deployment Toolkit

A complete, automated toolkit to remotely configure and clone Raspberry Pi 4 devices running LibreELEC.

🔐 SSH Setup
Install your key on all clients
💡 LED Disable
Silence all status LEDs
🔒 Profile Lock
Set Kodi profile PIN (default: 2737)
🗂 NAS Auto-Mount
Autostart NAS access
💾 Restore Backups
Drop in and extract full system configs
✅ GitHub Actions
Lints your scripts on push
